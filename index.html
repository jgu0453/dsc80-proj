League of Legends Esports Outcome Predictor

Introduction

This project analyzes a dataset from League of Legends esports matches to determine how key performance indicators (KPIs) influence game outcomes. We focus on the question: "How does gold difference at key timestamps affect win rates across different leagues?"

The dataset contains 40164 rows and includes columns such as "goldat15" (gold difference at 15 minutes), "result" (win/loss), and "league" (the league the match belongs to).

Framing Prediction Problem

We aim to predict game outcomes based on features such as "goldat15," champion picks, and first objectives. This is a binary classification problem, and we use accuracy as the evaluation metric to assess model performance.

Baseline Model

The baseline model uses features like "goldat15" and one-hot encoded champion picks. It achieved an accuracy of 69%, indicating room for improvement.

Final Model

The final model incorporates engineered features such as "gold_diff_per_minute" and "KDA," leveraging a Random Forest Classifier. Hyperparameter tuning improved accuracy to 97%.

Fairness Analysis

The model was evaluated for fairness based on the "side" (Blue vs. Red). Accuracy was consistent across groups (Blue: 96.7%, Red: 96.6%), suggesting no significant bias.
